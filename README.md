# Toybox Hugo Dot

## Getting Start

### Initialize Hugo Site
```bash
sh bin/init.sh
```

### Start Hugo server
```bash
cd site/bin
docker-compose up -d
```

## Create New document

### Create new section
```bash
cd site
sh bin/section <section name>
```

### Create content
```bash
sh bin/post <section name>/<post name>.md
```
